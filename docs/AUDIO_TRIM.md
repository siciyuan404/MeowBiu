# 音频裁切功能使用说明

## 功能概述

MeowBiu 应用现在支持完整的音频裁切和提取功能，可以帮助你：

- 🎵 **裁切音频** - 提取音频片段
- 🎬 **从视频提取音频** - 从视频文件中提取声音
- ✂️ **精确控制** - 设置开始和结束时间

## 使用方法

### 方式一：裁切现有音频

1. 在添加猫声页面，选择 **"裁切现有音频"**
2. 点击选择音频文件
3. 使用滑块设置裁切范围：
   - **开始滑块 (绿色)** - 设置片段开始位置
   - **结束滑块 (红色)** - 设置片段结束位置
4. 点击"预览"按钮试听选中的片段
5. 满意后点击"裁切音频"按钮
6. 裁切完成后保存即可

### 方式二：从视频提取音频

1. 在添加猫声页面，选择 **"从视频提取音频"**
2. 点击"选择视频文件"按钮
3. 选择需要提取音频的视频文件（支持 MP4、MOV、AVI、MKV 等格式）
4. 设置需要提取的时间范围
5. 点击"提取音频"按钮
6. 提取完成后保存

## 快速预设

为了方便使用，提供了以下快速预设按钮：

| 预设 | 说明 |
|------|------|
| 1s |裁取 1 秒片段 |
| 3s | 裁取 3 秒片段 |
| 5s | 裁取 5 秒片段 |
| 10s | 裁取 10 秒片段 |
| 一半 | 裁取前半部分 |
| 中间 | 裁取中间 5 秒 |
| 完整 | 恢复完整音频 |

## 精确模式

- **关闭（默认）**: 快速裁切，直接复制音频流，速度快
- **开启**: 精确裁切，重新编码音频，精度更高但速度较慢

## 技术实现

音频裁切功能基于 **FFmpeg** 实现：

- 使用 `ffmpeg_kit_flutter_audio` 库
- 支持的输入格式：MP3, WAV, OGG, AAC, M4A, FLAC, MP4, MOV, AVI, MKV
- 输出格式：MP3 (高质量)

## 注意事项

1. 裁切/提取过程可能需要几秒钟，请耐心等待
2. 裁切后的音频会保存在临时目录
3. 大文件或长视频可能需要更长的处理时间
4. 确保设备存储空间充足
